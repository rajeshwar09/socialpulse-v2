# SocialPulse V2

SocialPulse V2 is a production-style social media analytics project focused on YouTube data collection, lakehouse storage, sentiment analysis, forecasting, and prescriptive insights.

## Main goals

- Historical + daily incremental data storage
- Query-driven YouTube ingestion
- Bronze / Silver / Gold processing
- ML-based sentiment analysis
- Descriptive, Diagnostic, Predictive, and Prescriptive analytics
- Dark themed interactive dashboard

## Development model

The project is built phase by phase using long-lived Git branches.

## Planned phases

- phase-00-project-bootstrap
- phase-01-lakehouse-foundation
- phase-02-historical-bootstrap-json
- phase-03-query-driven-daily-ingestion
- phase-04-silver-enrichment-pipeline
- phase-05-ml-sentiment-engine
- phase-06-analytics-gold-layer
- phase-07-dashboard-v2-foundation
- phase-08-descriptive-diagnostic-dashboard
- phase-09-predictive-prescriptive-dashboard
- phase-10-production-polish
